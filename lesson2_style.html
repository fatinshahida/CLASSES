Writing CSS inline is easy and you know exactly which element the style will apply to. But this is not the preferred method because it mixes the content marked by HTML with the presentation done using CSS. Ideally, we want to keep those two separate. This is why you should try to place all your styles in an external style sheet. This doesn't mean that you should never use inline styles though!
